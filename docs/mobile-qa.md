# Mobile QA Notes

Use this checklist for narrow viewport and MiniPay webview reviews.

## Layout Checks

- Confirm dashboard cards remain readable on a 360px-wide viewport.
- Verify long wallet addresses truncate without pushing action buttons off screen.
- Check that transaction notices do not cover primary controls.

## Interaction Checks

- Tap through wallet connect, copy address, check-in, claim, and compound states.
- Confirm loading states clear after the wallet request is rejected or completed.
- Save the viewport width and device used for each mobile QA pass.
