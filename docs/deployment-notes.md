# Deployment Notes

Use these notes before promoting a CeloPulse preview to production.

## Preview Review

- Confirm the preview uses the intended branch and environment group.
- Run the same wallet smoke path in preview that will be used after promotion.
- Check that public metadata resolves from the preview URL.

## Production Review

- Compare production environment values against the preview values before promoting.
- Keep the previous production deployment URL available for rollback.
