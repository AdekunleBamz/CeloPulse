# Support Playbook

Use these notes when collecting enough context to diagnose a CeloPulse issue.

## Minimum Details

- Active URL, chain id, and wallet type used during the issue.
- Transaction hash for any completed or failed write action.
- Screenshot of the visible error state with private values removed.

## First Checks

- Confirm the production URL matches the configured app URL.
- Confirm the wallet is connected to the same chain as the configured contract.
